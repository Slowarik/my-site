Глобальные горячие клавши

1) Ctrl-O - Открыть файл
2) Ctrl+Z – Отменить последнее действие
3) Shift+Ctrl+Z – Вернуть действие назад (для пункта отмены)
4) Ctrl+N – Создать новую сцену
5) Tab – Переключение между режимами объекта и редактирования
6) Q - Быстрый доступ (избранное)

Общие клавиши редактирования

1)X - Удаляет выбранный элемент с диалогом подтверждения.
2)Delete - Удаляет выбранный элемент без диалога подтверждения.

Общие клавиши редактора

1)A - Выбрать всё.
2) Alt-A - Снять выделение.
3) Ctrl-I - Инвертировать выделение.
4) H - Скрыть выделенное.
5) Alt-H - Показать скрытые элементы.
6) T - Показать панель инструментов.
7) N - Переключение боковой панели.

Общие клавиши редактирования

Backspace - Удаляет выбранный элемент с диалогом подтверждения
Delete - Удаляет выбранный элемент без диалога подтверждения
Ctrl-D - Дублирование

Selection

1)ЛКМ - Select
2)Ctrl-A - Выбрать всё
3)Shift-Ctrl-A - Снять выделение
4)Ctrl-I - Обратить выбор
5)Up - Увеличить выделение
6)Down - Уменьшить выделение
